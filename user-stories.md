# User Stories

## Core query stories
- As a campaign manager, I can ask "what emails were sent to Military
  students last month" and get a formatted table of results
- As a campaign manager, I can ask about open rates for any segment
  and get the answer with benchmark context
- As a campaign manager, I can ask about SMS lift and get a comparison
  across audience segments

## Multi-agent stories
- As a campaign manager, I can ask "how do I improve performance for
  graduate students" and receive: current metrics, campaign ideas,
  AND have a project ticket automatically created in Airtable
- As a campaign manager, I can see which agents contributed to a
  response so I understand how the answer was produced

## Asset inventory stories
- As a campaign manager, I can ask what automated journeys are
  currently active and get a list with schedule details
- As a campaign manager, I can ask about any asset type (email,
  landing page, content block, SMS) in plain English

## Acceptance criteria (for each story)
- Response is factually correct against seed data
- Response is formatted appropriately (table, text, or both)
- Response completes within 15 seconds
- No SQL errors surface to the user
